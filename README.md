# Simple Banking System (Java)

A simple console banking application written in Java.

## Features
- Create a bank account with a unique ID
- Check account balance
- Deposit money
- Withdraw money
- Simple console menu

## Technologies
- Java
- HashMap (Java Collections)
- Scanner for user input
- Random for generating account IDs

## Example Menu

Available operations:
1. Create a bank account
2. Check balance
3. Deposit money
4. Withdraw money
5. Exit
